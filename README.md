# AI-Powered Patient Onboarding Hub

This capstone project focuses on developing an AI-powered patient onboarding hub within the Salesforce platform. The solution aims to streamline the initial patient registration and information gathering process, enhancing efficiency and patient experience in healthcare settings. It involves integrating various Salesforce functionalities, including Lightning components, Flow Builder, and Agentforce, to create an intelligent, automated, and secure system. The project emphasizes leveraging generative AI capabilities to personalize interactions and automate routine tasks associated with patient intake.
